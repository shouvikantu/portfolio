3) Reading Python (reimagined)

  • Traditional: trace code → describe output.
  • Here: given image(s) → write code to produce them.
  • Flags are fixed-pixel HTML (not image files) to resist copy→prompt tactics.
  • Discourages brute-force (49 manual tiles) via multiple regional variants.
This reverse-mapping cultivates structural reasoning: “What loop/offset produces this long horizontal color?” rather than “What does this loop do?”.
