Course context

  • CS151 Python (Stanford-inspired); projects drive learning.
  • First 3 “scripting” projects (1-file) average: 84, 87, 87%.
  • Multi-week “programming” projects drop to: 79% (Enigma), 69% (Adventure Game).
  • Interpretation: snippet-level prompting ≠ multi-file design & integration.

Assigned via GitHub Classroom; instructions in HTML; milestone pattern familiar to students.

Observation: Students strong on project scores sometimes could not write code unaided on the final
revealing over-reliance on code generation.
Milestone cadence: daily chunks, buffer day for debugging; same rhythm used for the final. This supports sustained engagement and reduces last-minute risk.
