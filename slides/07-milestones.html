Final structure & milestones

  1. Strings & Files — parse unusual data; choose representations.
  2. Interactive Graphics — generate structured image pattern.
  3. Reading Python — derive code from visual specs.
  4. Defining Classes — build an image-collage pipeline.
  5. Fundamental Python — vector ops & integer luminance.

Pipeline forces concept carryover; shallow/isolated solutions tend to fail later steps.

The cumulative design is intentional: inability to model early will surface in graphics and collage steps; numerical assumptions are tested in the luminance step.
